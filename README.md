# 劳务报酬计算器

一个基于Vue 3 + Vite开发的劳务报酬个税计算工具，支持正算和反算两种模式。

## 功能特点

- ✅ 劳务报酬预扣预缴个税计算
- ✅ 支持两种计算模式：
  - 输入劳务报酬，计算税后收入
  - 输入税后收入，反推劳务报酬
- ✅ 详细的计算结果展示，包括减除费用、应纳税所得额、适用税率、速算扣除数、应缴税款等
- ✅ 计算历史记录功能，最多保存10条记录
- ✅ 响应式设计，美观的UI界面
- ✅ 支持两位小数精度

## 技术栈

- **前端框架**：Vue 3 (Composition API)
- **构建工具**：Vite
- **样式**：原生CSS
- **开发语言**：JavaScript

## 计算规则

### 正算（从劳务报酬到税后收入）

1. **减除费用**：
   - 劳务报酬 ≤ 800元：减除费用为0
   - 800 < 劳务报酬 ≤ 4000元：减除费用800元
   - 劳务报酬 > 4000元：减除费用为劳务报酬的20%
2. **应纳税所得额**：
   - 劳务报酬 ≤ 800元：0
   - 800 < 劳务报酬 ≤ 4000元：劳务报酬 - 800
   - 劳务报酬 > 4000元：劳务报酬 × 80%
3. **税率表**（三级累进）：
   - 0-20000元：税率20%，速算扣除数0
   - 20000-50000元：税率30%，速算扣除数2000
   - 50000元以上：税率40%，速算扣除数7000
4. **应缴税款**：应纳税所得额 × 税率 - 速算扣除数
5. **税后收入**：劳务报酬 - 应缴税款

### 反算（从税后收入到劳务报酬）

根据不同的税率档位和减除费用规则，通过数学公式反推劳务报酬。
